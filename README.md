
## Galeria de Fotos

Este projeto foi desenvolvido como parte do desafio do processo seletivo da Aceleradora Ágil da PUCRS, conforme o Enunciado 02.

---

## Tecnologias Utilizadas

- **HTML5**
- **JavaScript**
- **Tailwind CSS**
---

## Como Executar

### Pré-requisitos
- Navegador web

### 1 - Clonar o repositório

```bash
git clone https://github.com/KristianKirschner/desafio-puc-enunciado-2.git
cd desafio-puc-enunciado-2
```

### 2 - Executar a aplicação
- Abra o arquivo `index.html` diretamente em um navegador web  
**ou**
- Utilize uma extensão como **Live Server** no VS Code para melhor experiência


## Funcionalidades

- Exibição de galeria de fotos em layout grid responsivo  
- Carregamento de um conjunto de imagens com títulos  
- Efeito visual ao passar o mouse sobre as fotos  
- Barra de busca para pesquisa por palavras-chave  
- Busca em tempo real conforme o usuário digita  
- Botão de pesquisa com ícone para execução da busca  
- Filtro das imagens considerando apenas o nome da foto  
- Exibição de mensagem quando nenhuma foto é encontrada
